# Mẫu Workflow cơ bản

Dưới đây là các bước quy trình mẫu bạn có thể tinh chỉnh sau:

1. Chạy cài đặt thư viện:
`npm install`

// turbo
2. Chạy kiểm tra mã nguồn (tự động chạy do có thẻ // turbo):
`npm run lint`

3. Build dự án:
`npm run build`
