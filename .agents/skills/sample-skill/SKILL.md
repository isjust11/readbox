# Hướng dẫn Coding Standard cho Readbox

Khi thực hiện code và yêu cầu tạo tính năng trên dự án Readbox, tôi (AI) sẽ đọc và luôn tuân thủ các quy tắc sau:

1. Luôn sử dụng cú pháp ES6+ (hoặc TypeScript nếu dự án quy định).
2. Tái sử dụng các ui components dùng chung nếu có trước khi tạo cái mới.
3. Khi viết logic lấy dữ liệu, nhớ xử lý các trường hợp lỗi (try/catch).
4. Viết file test nếu cần thiết.
5. (Thêm các quy tắc đặc thù khác của team bạn tại đây...)
